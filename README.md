# Ipymafs - Interactive math react components in jupyter

Still work in progress.

Brings https://mafs.dev/ by @stevenpetryk to jupyter.

build ontop of anywidget.

```
pip install ipymafs
```

Example script:

```python
from ipymafs import Bezier
b = Bezier(p1x = -5)
b
```

```
b.p1x = 4
```
